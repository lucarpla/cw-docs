# How to Disable Printing Option Set Name

### Steps:

1. Log in to your account[ https://admin.Rakoon.com/login](https://admin.Rakoon.com/login)
2. Go to **Menus** > **Option Sets**

<figure><img src="../.gitbook/assets/unknown (24) (1) (1).png" alt="" width="563"><figcaption></figcaption></figure>

3. Open the option set you don’t want to print by clicking the **Edit** icon.

<figure><img src="../.gitbook/assets/unknown (25) (1) (1).png" alt="" width="563"><figcaption></figcaption></figure>

4. Under **General**, toggle on **Disable Printing Option Set Name.**

<figure><img src="../.gitbook/assets/unknown (26) (1) (1).png" alt="" width="563"><figcaption></figcaption></figure>

5. Click **Save.**
